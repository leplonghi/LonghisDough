# Component Inventory

## Pages
- **CalculatorPage**: Main utility. Renders `CalculatorForm` and `ResultsDisplay`. Handles onboarding tooltip logic.
- **MyLabPage**: Dashboard. Shows greeting, Levain card, Last Batch card. Checks Pro status.
- **CommunityPage**: Displays feed of batches fetched from LocalStorage (`communityStore`).
- **DoughStylesPage**: Grid of style cards. Opens `StyleDetailPage` on click.
- **LearnPage**: Grid of learning topics.
- **ShopPage**: Displays affiliate products.
- **ProfilePage**: User settings, Ovens list, Levains list.

## Core Components
- **CalculatorForm**: Complex form with sliders/inputs. Handles ingredients, pre-ferments, environment. Contains `FlourBlendEditor` and `IngredientTableEditor`.
- **ResultsDisplay**: Shows calculated weights. Contains `RecipeTimeline` and `SuggestionsBlock`.
- **UserMenu**: Dropdown for profile/settings/logout.
- **Navigation**: Responsive header + sidebar (in MyLab) + MobileSummaryBar.

## Logic Components
- **FlourBlendEditor**: Allows mixing multiple flours and calculating estimated W/Protein.
- **IngredientTableEditor**: Allows adding custom ingredients (eggs, milk, etc.).
- **RecipeTimeline**: Visual vertical timeline of recipe steps generated by logic.
- **LevainInsights**: Visualizes health score and history of a Levain.

## Modals
- **PaywallModal**: Upsell screen. Triggers `grantProAccess`.
- **LevainModal**: Create/Edit levain.
- **OvenModal**: Create/Edit oven.
- **ImportReceitaModal**: Import JSON recipe.
- **GoalModal**: Create/Edit goals.
- **ConsistencySeriesModal**: Create/Edit test series.
